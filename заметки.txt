#  Модель пользователя Django включает в себя следующие поля:
#
# username — Никнейм. Обязательное поле. Может содержать до 150 символов, включая  _, @, +, . и -. Именно по этому полю, например, по e-mail в Django работает аутентификация по умолчанию.
# first_name — Имя. Необязательное поле.
# last_name — Фамилия. Необязательное поле.
# e-mail — Электронный адрес. Необязательное поле.
# password — Пароль. Обязательное поле. В этом поле будет храниться не сам пароль, а его хэш и метаданные. Django не хранит вводимые пользователями пароли в открытом виде.
# is_staff — Булево поле. Определяет, имеет ли пользователь доступ к админке сайта (/admin).
# is_active — Определяет, является ли пользователь активным. Например, может ли он залогиниться и проводить какие-то действия на сайте.
# is_superuser — Определяет, имеет ли пользователь все права, без явного их перечисления.
